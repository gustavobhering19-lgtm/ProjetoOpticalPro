
# Optical Pro

## 🚀 Demonstração
Projeto em desenvolvimento. Em breve disponível online.

## 📌 Funcionalidades
- Cálculo de espessura de lentes.
- Validação de formulários.
- Interface responsiva.
- Visualização gráfica das lentes.

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## 📂 Estrutura do Projeto

```text

ProjetoOpticalPro/
│
├── index.html          # Estrutura principal da aplicação
├── app.js              # Inicialização e controle principal
├── ui.js               # Manipulação da interface
├── validation.js       # Validação dos dados inseridos
├── Lensmath.js         # Cálculos ópticos e espessura das lentes
├── Lensvisual.js       # Visualização gráfica das lentes
│
├── variables.css       # Variáveis globais
├── base.css            # Estilos base
├── layout.css          # Estrutura do layout
├── components.css      # Componentes reutilizáveis
├── form.css            # Estilos dos formulários
├── results.css         # Exibição dos resultados
├── animations.css      # Animações da interface
├── responsive.css      # Responsividade
│
├── README.md
└── LICENSE

```

## 🎯 Objetivo

O Optical Pro foi desenvolvido com o objetivo de auxiliar profissionais do ramo óptico na estimativa da espessura de lentes oftálmicas, proporcionando maior agilidade e praticidade no atendimento ao cliente.

Além de sua aplicação prática, o projeto também foi criado como parte do meu processo de aprendizado em desenvolvimento web, com foco na aplicação de conceitos de HTML, CSS e JavaScript, organização de código, manipulação do DOM, validação de formulários e criação de interfaces responsivas.

Este projeto representa minha evolução como desenvolvedor, unindo conhecimentos técnicos à minha experiência profissional no setor óptico para criar uma solução útil e alinhada às necessidades do mercado.
